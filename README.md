# About
S-PONG is a 2D Pong Game with space theme created with Unity Engine.

# Gameplay
Players control spaceship positioned on opposite sides of the screen, maneuvering them vertically or horizontally to hit a comet-like ball back and forth. The goal is to score points by getting the ball past the opponentâ€™s paddle. The simplicity of the controls makes it accessible, while the increasing ball speed as the rally continues keeps the challenge alive.

- **Local Multiplayer** <br>
  Enjoy local multiplayer, facing off against your friend.
